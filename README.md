# Multi-Paradigm Problem Solving: Statistics Calculator

## Overview
This project implements mean, median, and mode using:
- C (Procedural)
- OCaml (Functional)
- Python (Object-Oriented)

## Structure
C/, OCaml/, Python/, screenshots/, report/

## Run
C: gcc statistics.c -o stats && ./stats  
OCaml: ocaml statistics.ml  
Python: python statistics.py
